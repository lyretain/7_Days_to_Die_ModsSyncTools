配置文件格式
```json
{
    "program_name": "7 Days to Die", //游戏名称
    "mods_folder": "\\Mods\\", //文件同步子目录，游戏安装目录\文件同步子目录\
    "server_url": "" //服务器文件链接
}
```
服务器文件格式
```json
[
  {
    "name": "Localization.txt", //文件名称
    "path": "A21-现代食物-稳定版\Config\Localization.txt", //文件路径
    "link": "http://localhost/7DaysToDie/A21-现代食物-稳定版/Config/Localization.txt" //服务器端文件链接
  }
]
```
